TestFlight

Install AI-Cabulary before the App Store release.

TestFlight is the first release path for iPhone, iPad, and Mac while the app is still being validated.

What testers need

  • An Apple device running a supported OS version.
  • The TestFlight app installed from Apple.
  • An invite link or email from the AI-Cabulary team.

Install flow

  1. Open the TestFlight invitation on your device.
  2. Accept the invite and install the latest AI-Cabulary build.
  3. Launch the app and allow camera access only if you want OCR from photos or camera capture.
  4. On first model preparation, wait for the local model to become ready.

What to test

  • Adding vocabulary manually.
  • Reading context from library images, camera images, and pasted images.
  • Translation quality for different language pairs.
  • Practice flow and mastered-card behavior.
  • Startup model preparation on supported hardware.
